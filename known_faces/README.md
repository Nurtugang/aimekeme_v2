# known_faces — мини-база лиц

Сюда кладём по одному фото на человека. Имя файла = имя человека.

```
known_faces/
├── nurtugan.jpg
├── aybek.jpg
└── ...
```

Правила:
- один файл = один человек, на фото **одно чёткое лицо** анфас;
- форматы: `.jpg`, `.jpeg`, `.png`;
- имя файла (без расширения) становится `identity` в ответе API.

При старте сервиса для каждого фото считается эмбеддинг (512-мерный вектор).
Входящее лицо сравнивается со всеми по косинусной близости; если ближайшее
>= `FACE_MATCH_THRESHOLD` (по умолчанию 0.6) — возвращается это имя, иначе `unknown`.

Добавил/удалил фото — **перезапусти сервис** (база строится на старте).
